Por aquí proyectos que he desarrollado mediante la utilización de diferentes tecnologías, escalables y legibles. No solo para poder demostrar mis conocimientos, sino por la motivación tanto de crear como aportar.

EcoStream

Plataforma avanzada de telemetría urbana que procesa datos ambientales en tiempo real. La arquitectura se basa en un modelo dirigido por eventos mediante Apache Kafka, garantizando una alta concurrencia y escalabilidad, apoyada por Redis para una gestión de caché ultrarrápida y persistencia de estados temporales. Además, integra un agente inteligente basado en IA (Groq/Llama 3.1), que realiza un análisis cognitivo profundo y auditoría automatizada de los datos, optimizando la toma de decisiones.

FastAPI React Kafka PostgreSQL Redis Tailwind Python Lite

SIAntesis

Plataforma avanzada de debate analítico y toma de decisiones que emula la deliberación humana mediante un ecosistema multi-agente en LangGraph. El núcleo confronta de forma cíclica un enfoque idealista (Estratega) y uno realista (Auditor), procesado mediante inferencia de ultra baja latencia con Groq (Llama 3.3). La interfaz ofrece una experiencia interactiva en pantalla dividida sincronizada en tiempo real mediante Server-Sent Events (SSE), e integra memoria semántica histórica a largo plazo basada en búsquedas de similitud vectorial utilizando PostgreSQL (PgVector).

FastAPI React LangChain PgVector Groq Tailwind Docker CI/CD

Pomodoro Timer

Aplicación de productividad de alto rendimiento diseñada bajo un enfoque de arquitectura modular y escalable. Desarrollada con React y Vite para ofrecer una interfaz fluida y una respuesta ultrarrápida. La aplicación integra un pipeline de CI/CD mediante GitHub Actions, garantizando la validación automática del código y un despliegue continuo eficiente. El entorno de desarrollo está estandarizado mediante Docker, asegurando la paridad entre el entorno de desarrollo y producción. Enfocada en la gestión eficiente del tiempo, integra Tailwind CSS para una UI consistente, moderna y altamente responsiva.

React Vite Docker MySQL Tailwind

Spotic

Réplica funcional de Spotify conectada a su API oficial. Cuenta con un sistema de contingencia (Resilience Fallback): debido a las novedosas restricciones estrictas que Spotify for Developer ha implementado para cuentas no Premium, el backend conmuta automáticamente a un entorno de demostración enriquecido para garantizar el 100% de la navegación.
